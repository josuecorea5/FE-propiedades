# Frontend for Managing Properties App

## Overview
This is the frontend for the Managing Properties app. It allow to users to manage their properties, contact with other users for their properties.

## Technologies
- React
- Nextjs 13
- Typescript
- Tailwindcss
- React hook form
- Pigeon maps
- LocationIQ
- Zod
- Sweetalert2
